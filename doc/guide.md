1.  순수 html, tailwindcss, javascript 조합 + cdn 으로 라이브러리 사용
2. 사용자는 경주달팽이 개수(필수), 색깔(안넣으면 임의 배정)과 이름표(안넣으면 임의 배정), 타입(필수)을 정하고 게임 시작을 한다
2. 시소가 보이며 좌측 부터 우측까지 경주를 시작한다 . 상단의 자신 달팽이 색깔, 이름표 배정
3. 달팽이 속도 및 경주 알고리즘은 문서를 참조한다

1. 각종 효과들이 필요하며 달팽이가 빠를때 느릴때, 시소의 영향을 받을때 등등이 적절이 들어가야 한다
2. ui 는 화려할 필요가 없으며 복잡하지 않아 보야야 한다
3. 상단 45 도에서 경기장이 보이는 방식으로 시소(경기장)가 보여야 하고 미끌미끌 달팽이는 더 미끌미끌하도록 보여야함, 빤딱빤딱 달팽이는 약간 더 빤딱빤딱하게 보여야 함, 달팽이 진액이 뒤에 남도록 해야함
