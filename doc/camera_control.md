# 카메라 제어

우측 하단은 카메라 제어용 ui
우측 하단에 2가지의 ui 가 있음 

### 카메라 자유도 보장용

이동 및 회전용(구형(회전용) 중앙에 상하좌우 버튼(이동용)), 확대용 슬라이더
구형이 중앙에 위치하고 상하좌우 버튼은 (구형은 돌아가는 느낌이 있어야함)
확대 제어 슬라이더 (Velocity Control Slider)
3가지는 화면에 클릭 이동등 다른 로직들과 동기화되어 돌아감

잡기 기능 정의 : left mouse button down 이동 left mouse button up

이동 : shift+마우스 잡기
회전 : 마우스 잡기
확대 : 마우스 스크롤


### 자유도 제한 프리셋
3가지 프리셋
기본 화면용 카메라 배치
경주 시작에서 경주 끝을 보는 프리셋 (출발선)
경주 끝에서 시작을 보는 프리셋 (결승선)
이미 좌측 상단에 달팽이 통계들이 있는데 거기의 각 달팽이를 누르면 각 달팽이 시점에서 볼수 있는 카메라

해당 4가지 프리셋은 각 객체에 따라다니면서 동작하는 것임
달팽이 시점회전 임 즉 사용자의 의도적 `이동`을 제외하고 회전, 확대는 그대로 존중해야함
